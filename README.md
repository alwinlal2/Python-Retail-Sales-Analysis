
# Retail Sales Data Cleaning & Exploratory Data Analysis (EDA)

## Project Overview

This project demonstrates an end-to-end data analysis workflow using Python and Pandas. The dataset contained several real-world data quality issues, including missing values, inconsistent formats, duplicate records, invalid dates, incorrect calculations, and unrealistic values.

The objective was to clean the dataset, validate the integrity of important business metrics, and perform exploratory data analysis (EDA) to extract meaningful insights.

---

## Objectives

- Clean messy retail sales data
- Validate business calculations
- Handle missing and inconsistent values
- Explore sales trends
- Analyze customer behaviour
- Generate business insights using visualizations

---

## Tools Used

- Python
- Pandas
- NumPy 
- Matplotlib
- Jupyter Notebook

---

## Data Cleaning Performed

The dataset was cleaned by performing the following tasks:

- Removed duplicate records
- Handled missing values
- Standardized date formats
- Converted currency strings into numeric values
- Corrected data types
- Removed invalid quantity and discount values
- Verified sales calculations using Quantity × Unit Price × (1 − Discount)
- Created additional columns where required
- Saved a cleaned version of the dataset
